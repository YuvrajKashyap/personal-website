# Hero Source Assets

Source hero images live in this folder.

Optimized production assets live in `public/media/hero`.

Source assets may be larger than production assets. They are kept so poster images can be regenerated consistently.

Do not bake text, UI panels, navigation, or buttons into source images. Future hero text, CTAs, orbital UI, and panels must remain coded UI.

Do not replace the selected source without architect approval.

If a raw source asset is extremely large, document the size before committing it.
